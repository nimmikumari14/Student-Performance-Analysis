# Student Performance Analysis

## Project Objective
Analyze student academic performance and identify factors affecting scores.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- Female students performed better in reading and writing.
- Male students performed slightly better in mathematics.
- Test preparation improved scores.
- Several students were identified as at-risk.
- Reading and writing scores showed strong correlation.
